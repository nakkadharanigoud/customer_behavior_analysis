# 📊 Customer Behavior Data Analytics Portfolio Project

---

## 🎯 Project Overview

This project demonstrates a complete **end-to-end Data Analytics workflow** designed to simulate real-world business analysis scenarios.

### 🔹 Objectives

* Clean and prepare raw customer data
* Perform Exploratory Data Analysis (EDA)
* Load data into PostgreSQL
* Write SQL queries to answer business questions
* Build an interactive Power BI dashboard
* Generate business insights and recommendations

## 📂 Project Workflow

### 1️⃣ Data Collection & Understanding

* Imported dataset using Pandas
* Examined data structure and features
* Identified data quality issues

### 2️⃣ Data Cleaning & Preprocessing

* Handled missing values
* Removed duplicate records
* Corrected inconsistent entries
* Standardized data formats

### 3️⃣ Exploratory Data Analysis (EDA)

#### 👥 Customer Analysis

* Customer demographics
* Purchase patterns
* Spending behavior
* Product preferences
* Customer segmentation

#### 📊 Visualization

* Created charts using Matplotlib and Seaborn
* Identified trends and patterns
* Performed statistical analysis

### 4️⃣ Database Integration

* Created PostgreSQL database
* Connected Python with PostgreSQL
* Loaded cleaned dataset into SQL tables

### 5️⃣ Business Analysis Using SQL

* Which customer segments generate the highest revenue?
* What are the most popular product categories?
* Which age groups spend the most?
* How does shopping frequency impact sales?
* What factors influence customer purchasing behavior?

### 6️⃣ Power BI Dashboard Development

* 📈 Revenue KPIs
* 👥 Customer Demographics Analysis
* 🛒 Product Category Performance
* 📊 Sales Trends
* 🎯 Customer Segmentation
* 🔍 Interactive Filters & Slicers
